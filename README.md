#  Rock Paper Scissors Game (Python)

A simple command-line **Rock Paper Scissors** game built using Python. Challenge the computer in a **best-of-three** match, where each round is decided based on the classic game rules.

---

##  Features

-  Play against the computer.
-  Computer makes random choices using Python's `random` module.
-  Best-of-three gameplay.
-  Keeps track of player and computer scores.
-  Input validation for invalid choices.
-  Option to play multiple games.
-  Beginner-friendly and easy-to-understand code.

---

##  Technologies Used

- Python 3
- random module
- PyCharm

##  Game Rules

-  Rock beats Scissor
-  Scissor beats Paper
-  Paper beats Rock
- The game consists of **3 rounds**.
- The player with the highest score after three rounds wins.
